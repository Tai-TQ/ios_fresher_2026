# Fresher2026

Demo repo for the **iOS Fresher 2026** course, built with Swift + UIKit (programmatic UI).

## Requirements
- Xcode 15+, iOS 15+

## Run
Open `Fresher2026.xcodeproj`, pick the `Fresher2026` scheme and a simulator, then press `Cmd + R`.

## Navigation
`ViewController` shows a vertical stack view of buttons; each one pushes a demo screen. Add a new demo by appending a button in `addDemoButtons()`. Current demos: **Table View** (`UITableView` with a custom cell).
